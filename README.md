# xiaohongshu-post

小红书文案发布 Skill - 使用子Agent协作流程发布内容到小红书。

## 概述

使用 6 个子 Agent 协作流程，人机协作模式。支持：

## 特点

- 🆓 **零成本** - 不用付费API，浏览器直接操作
- 🔒 **安全不封号** - 人机协作，每步确认
- 👶 **小白友好** - 不用写代码，有手就会
- ⚡ **人机协作** - 人确认 + AI执行，比全自动更安全
- 🌐 **浏览器操作** - 无需API Key，不花额外费用
- 🔄 **灵活切换** - 想省钱用浏览器，想更自动化可换API

## 支持
- 纯文字（长文）
- 图文（图片 + 文字）
- 文字配图（小红书AI生成卡片）

## 适用场景

- "发小红书"
- "写一篇小红书"
- "发布到小红书"
- "帮我发小红书文案"

## 前置要求

1. OpenClaw 环境配置完成
2. 子 Agent 配置：researcher, writer, executor, analyst
3. 飞书多维表格（用于素材管理，可选）
4. 小红书创作者账号

## 工作流程

### 1. researcher 搜热点（可选）
- 搜索今日热点选题，确定写作方向
- 如果用户已有明确主题，可跳过

### 2. writer 写文案
- 从飞书多维表格拉取素材（可选）
- 创作小红书风格文案

### 3. analyst 优化
- 检查文案质量
- 优化标题和标签

### 4. 用户确认
- 将文案发给用户，等待确认

### 5. executor 发布
- 打开小红书发布页
- 辅助完成发布

### 6. 确认成功
- 检查发布结果
- 记录到飞书（可选）

## 发布方式

### 方式 A：文字配图（推荐）
1. 打开 creator.xiaohongshu.com
2. 发布笔记 → 上传图文 → 文字配图
3. 输入文案 → 生成图片
4. 选择样式 → 下一步 → 发布

**优点**：不需要自己找图，小红书自动生成卡片图

### 方式 B：上传图片
1. 准备图片素材
2. 上传图片 + 文字
3. 发布

### 方式 C：长文
1. 写长文
2. 输入标题和正文
3. 发布

## 配置飞书素材库（可选）

创建飞书多维表格管理素材：

| 表格 | 用途 |
|------|------|
| AI成长经历库 | 原始素材 |
| 文案脚本库 | 写好的文案 |
| 自媒体选题库 | 待创作选题 |
| 已发布内容库 | 已发布记录 |

## 已知问题

### Windows 文件选择弹窗
- Chrome 扩展无法控制 Windows 系统级文件选择对话框
- 解决方案：使用"文字配图"功能，或用户手动选择图片
