# 小红书文案发布 Skill

个性化、定制化小红书发布技能。使用子Agent协作流程。

## 概述

使用 6 个子 Agent 协作流程，人机协作模式。支持：
- 纯文字（长文）
- 图文（图片 + 文字）

## 特点

- 🆓 零成本 - 不用付费API，浏览器直接操作
- 🔒 安全不封号 - 人机协作，每步确认
- 👶 小白友好 - 不用写代码，有手就会
- ⚡ 人机协作 - 人确认 + AI执行，比全自动更安全
- 🌐 浏览器操作 - 无需API Key，不花额外费用

## 适用场景

- "发小红书"
- "写一篇小红书"
- "发布到小红书"
- "帮我发小红书文案"

## 前置要求

- OpenClaw 环境配置完成
- 子 Agent 配置：信息员、内容策划师、文案编辑师、视觉设计师、运营执行专员
- 飞书多维表格（用于素材管理，可选）
- 小红书创作者账号

## 工作流程

1. 信息员搜素材（全网搜索）
2. 确认素材（要不要加入飞书）
3. 内容策划师出大纲
4. 确认参考素材
5. 文案编辑师写文案
6. 确认文案（OK吗？方向对吗？）
7. 视觉设计师配图
8. 运营执行专员发布

## 子 Agent

| Agent | 名称 | 任务 |
|-------|------|------|
| researcher | 信息员 | 搜索素材 |
| content_planner | 内容策划师 | 出大纲 |
| copy_editor | 文案编辑师 | 写文案 |
| visual_designer | 视觉设计师 | 配图/封面 |
| operations | 运营执行专员 | 发布 |
