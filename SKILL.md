# 小红书发布 Skill

## 确认环节

问用户想发布什么类型：
1. 笔记（文字配图）
2. 长文
3. 视频

## 发布流程

### 笔记（文字配图）
1. 打开发布页 → 点击"文字配图"（不要点上传图文！）
2. 粘贴文案
3. 等待AI生成配图
4. **截大图给用户选样式** → 用户确认
5. 如需替换图片：点"图片编辑" → 点图片左下角的"+"按钮 → 选择本地图片
6. **用户确认后** → 下一步 → 发布

### 长文
1. 打开发布页 → 写内容
2. 设置标题话题 → 发布

### 视频
1. 打开发布页 → 上传视频
2. 填写标题描述 → 添加话题 → 发布

## 关键提醒

- **只用"文字配图"**，不要用上传图文（会弹窗无法控制）
- 点"文字配图"后直接输文案，不需要上传图片
