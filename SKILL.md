# 小红书发布技能（文字配图版）

## 概述
供运营执行专员使用的小红书发布技能，完整的人机协作发布流程。

## 完整流程

一、打开发布页
二、选择「上传图文」
三、输入文案
四、点「文字配图」生成配图
五、截图发给用户，选喜欢的卡片风格
六、用户选好后，点下一步
七、粘贴文案、标题
八、截图发给用户确认（客户可手动添加图片）
九、客户确认后，点「暂存离开」或「发布」

## 关键规则

- 必须严格按顺序执行，每步拿到结果后才能下一步
- 发布前必须用户预览确认，客户说可以才能点发布
- 客户说可以才能点发布，没说就继续等待
- 账号用当前登录的账号
- 每步结果及时反馈给调度者

## 结果要求

每步必须拿到明确结果：
- 截图发给用户 → 等待用户回复
- 用户确认后才能进行下一步
- 完成后反馈：发布成功 / 已存草稿 / 失败原因
