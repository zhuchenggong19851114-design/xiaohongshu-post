# 小红书文案发布 Skill

## 概述

使用 6 个子 Agent 协作流程，人机协作模式。支持：
- 纯文字（长文）
- 图文（图片 + 文字）

## 触发方式

当用户要求：
- "发小红书"
- "写一篇小红书"
- "发布到小红书"
- "帮我发小红书文案"

## 工作流程（必须按顺序执行）

### 步骤 1：信息员搜索素材
- **Agent**：信息员
- **任务**：全网搜索类似主题的资料
- **工具**：tavily-search, web_fetch
- **返回**：相关资料摘要

### 步骤 2：确认素材
- **确认**：要不要加入我们的素材（飞书素材库）

### 步骤 3：内容策划师出大纲
- **Agent**：内容策划师
- **任务**：根据素材出大纲
- **返回**：内容大纲

### 步骤 4：确认参考素材
- **确认**：有没有参考素材/小红书内容？

### 步骤 5：文案编辑师写文案
- **Agent**：文案编辑师
- **任务**：根据大纲写小红书风格文案
- **工具**：feishu_doc, feishu_bitable
- **飞书拉素材**：从飞书多维表格拉取素材

### 步骤 6：确认文案
- **确认**：文案OK吗？方向有没有偏差？

### 步骤 7：视觉设计师配图
- **Agent**：视觉设计师
- **任务**：做封面/配图
- **工具**：browser, image

### 步骤 8：运营执行专员发布
- **Agent**：运营执行专员
- **任务**：打开小红书发布页，辅助发布
- **工具**：browser, message

## 发布方式

### 方式 A：图文发布（推荐）
1. 打开 https://creator.xiaohongshu.com/publish/publish?target=image
2. 点击"上传图文"
3. 用户手动选择图片
4. 粘贴文案
5. 点击发布

### 方式 B：文字配图（自动生成）
1. 打开 https://creator.xiaohongshu.com/publish/publish?target=image
2. 点击"文字配图"
3. 输入文案，自动生成配图
4. 选择样式，点击发布

### 方式 C：长文发布
1. 打开 https://creator.xiaohongshu.com/publish/publish?target=text
2. 写长文
3. 设置标题、话题
4. 点击发布

## 子 Agent 配置

| Agent | 名称 | 任务 | 工具 |
|-------|------|------|------|
| researcher | 信息员 | 搜索素材 | tavily-search, web_fetch |
| content_planner | 内容策划师 | 出大纲 | - |
| copy_editor | 文案编辑师 | 写文案 | feishu_doc, feishu_bitable |
| visual_designer | 视觉设计师 | 配图/封面 | browser, image |
| operations | 运营执行专员 | 发布 | browser, message |

## 输出格式

**必须先发给用户确认：**
```
📕 小红书文案（复制粘贴）

---
标题：xxx

正文：xxx

#标签1 #标签2 #标签3
```

## 重要提醒

1. 必须先把文案发给用户，等用户确认后，再发布！
2. 图片上传需要用户配合（当前技术限制）
3. 浏览器在 WSL2 环境中，关闭浏览器会丢失登录状态
